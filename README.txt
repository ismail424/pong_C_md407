# Pong Game for  MD407

Simple pong game for MD407 board using STM32F407VG microcontroller.
The game uses two 4x4 keypads on GPIO D and LCD/Display on GPIO E.
The project also uses md407 library that can be found [here](https://github.com/pythar-chalmers/libmd407)
